# 技术栈

## 框架与运行时


| 技术             | 版本      | 用途         |
| -------------- | ------- | ---------- |
| **Next.js**    | 16.1.1  | React 全栈框架 |
| **React**      | 19.2.3  | UI 库       |
| **TypeScript** | 5.9.3   | 类型系统       |
| **pnpm**       | 10.15.0 | 包管理器       |


## AI / LLM 相关


| 技术                     | 用途              |
| ---------------------- | --------------- |
| **Mastra** (beta)      | AI Agent 框架     |
| **@ai-sdk/react**      | AI SDK React 集成 |
| **@ag-ui/client**      | Agent UI 组件     |
| **CopilotKit**         | AI Copilot 集成   |
| **ollama-ai-provider** | Ollama 本地 LLM   |
| **@ai-sdk/deepseek**   | DeepSeek 模型     |

## 总结

这是一个 **Next.js 16 全栈 AI 演示/落地页项目**，集成了：

- 🤖 **Mastra** AI Agent 框架
- 💬 **CopilotKit** AI 对话界面
- 🎨 **GSAP + Framer Motion** 高级动画
- 🌐 **Lingui** 国际化
- 📱 **Tailwind CSS 4** + **shadcn/ui** 现代 UI
